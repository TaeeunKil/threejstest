# 현재 작업 맥락

## 현재 작업 포커스

프로젝트 완성되었습니다. 6축 로봇 팔 시각화 데모가 작동 중입니다.

## 최근 변경사항

- 프로그래밍 방식으로 6축 로봇 팔 모델 생성 완료
- 계층적 관절 구조 구현 (부모-자식 관계로 정확한 회전)
- 관절 제어 UI 구현 (슬라이더 + 직접 입력)
- 반응형 레이아웃 구현 (데스크톱/모바일)
- 터치 제스처 지원 추가
- 불필요한 파일 정리 (모델 파일, 사용하지 않는 유틸리티)
- 프로젝트 이름 변경 (Rainbow Robotics → Three.js Demo)
- 최종 커밋 완료

## 다음 단계

현재는 완성된 상태입니다. 필요시 추가 개선 사항:
- TCP 위치 수치 표시
- 역기구학(Inverse Kinematics) 구현
- 더 많은 시각적 효과 추가
- 애니메이션 기능 추가

## 활성 결정사항

- 외부 모델 파일 대신 프로그래밍 방식으로 로봇 생성
- Three.js Group 계층 구조 활용
- 관절 회전축에 따라 원통 방향 조정
- 눕혀진 관절(Shoulder, Elbow, Wrist2)의 회전 중심을 관절 중심으로 설정
- 데스크톱: 슬라이더 패널 300px 고정, 모바일: 하단 배치

## 중요한 패턴 및 선호사항

- Three.js 계층 구조를 활용한 정확한 관절 회전
- 각 관절마다 좌표계 표시
- 반응형 UI로 모든 기기 지원
- 깔끔한 코드 구조와 명확한 주석

## 학습 및 프로젝트 인사이트

- Three.js에서 계층 구조가 매우 중요함
- Group의 position이 회전 중심이 됨
- OrbitControls는 터치 제스처를 기본 지원
- Flexbox로 반응형 레이아웃 구현이 효과적
- 불필요한 파일 정리가 프로젝트 유지보수에 중요
