# 현재 작업 맥락

## 현재 작업 포커스

프로젝트 초기 설정 단계입니다. React + Vite로 프로젝트를 세팅하고 있습니다.
작업은 커밋 단위로 진행됩니다.

## 최근 변경사항

- Memory Bank 구조 생성 및 업데이트
- .cursorrules 파일 설정
- 프로젝트 목적 및 컨벤션 정의
- React + Vite 프로젝트 초기 설정 완료 (첫 번째 커밋)

## 다음 단계

- Three.js 기본 씬 구현 (두 번째 커밋)
- 6축 로봇 팔 3D 모델 생성
- 관절 각도 조절 UI 구현
- 좌표계 및 TCP 표시

## 활성 결정사항

- React + Vite 사용 결정
- Three.js 최신 버전 사용
- 커밋 단위로 작업 진행
- 코드 컨벤션 및 커밋 컨벤션 정의 완료

## 중요한 패턴 및 선호사항

- 명확한 코드 구조
- 주석을 통한 학습 내용 문서화

- 의미 있는 커밋 메시지 작성
