# 제품 컨텍스트

## 프로젝트가 존재하는 이유

레인보우로보틱스 면접 포트폴리오로, 실제 로봇 제어 UI와 유사한 인터페이스를 Three.js로 구현하여
3D 시각화 및 프론트엔드 개발 역량을 증명하기 위함입니다.

## 해결하는 문제

- 면접관에게 Three.js 3D 시각화 경험을 시각적으로 보여줌
- 로봇 제어 UI의 복잡한 인터페이스를 웹에서 구현하는 능력 증명
- WebGL, Three.js 등 우대조건 기술 스택 경험 증명

## 작동 방식

- React + Vite로 모던 프론트엔드 프로젝트 구성
- Three.js를 사용하여 6축 로봇 팔을 3D로 렌더링
- 슬라이더를 통해 각 관절의 각도를 조절
- 실시간으로 로봇 팔의 자세가 업데이트됨
- 좌표계와 TCP 위치를 시각적으로 표시

## 사용자 경험 목표

- 실제 레인보우로보틱스 UI와 유사한 다크 테마 디자인
- 직관적인 로봇 팔 조작 인터페이스
- 부드러운 3D 애니메이션 및 인터랙션
- 면접관이 쉽게 이해할 수 있는 명확한 구조
