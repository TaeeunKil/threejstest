# 제품 컨텍스트

## 프로젝트가 존재하는 이유

Three.js를 사용한 6축 로봇 팔 시각화 데모로, 3D 웹 그래픽과 인터랙티브 UI 구현 능력을 보여주기 위한 프로젝트입니다.

## 해결하는 문제

- Three.js 3D 시각화 기술 데모
- 복잡한 계층 구조를 가진 로봇 팔 모델링
- 실시간 인터랙티브 제어 구현
- 데스크톱과 모바일 모두에서 작동하는 반응형 UI

## 작동 방식

- React + Vite + TypeScript로 모던 프론트엔드 프로젝트 구성
- Three.js를 사용하여 6축 로봇 팔을 프로그래밍 방식으로 생성
- 계층적 Group 구조로 부모-자식 관계 설정하여 관절 회전 구현
- 슬라이더와 직접 입력으로 각 관절의 각도를 조절
- 실시간으로 로봇 팔의 자세가 업데이트됨
- 각 관절마다 좌표계를 표시하여 회전 방향 시각화
- OrbitControls로 3D 뷰 조작 가능

## 사용자 경험 목표

- 다크 테마의 현대적인 UI 디자인
- 직관적인 로봇 팔 조작 인터페이스
- 부드러운 3D 인터랙션
- 데스크톱: 좌우 분할 레이아웃
- 모바일: 상하 분할 레이아웃
- 터치 제스처로 3D 뷰 조작
