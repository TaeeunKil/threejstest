# 프로젝트 개요

## 프로젝트 이름

threejs-demo

## 설명

Three.js를 사용한 6축 로봇 팔 시각화 데모 프로젝트입니다.
프로그래밍 방식으로 생성된 로봇 팔 모델과 인터랙티브한 제어 UI를 제공합니다.

## 목표

- Three.js를 활용한 6축 로봇 팔 3D 시각화 구현
- 프로그래밍 방식으로 로봇 팔 모델 생성
- 인터랙티브한 관절 제어 인터페이스 개발
- 반응형 UI로 데스크톱과 모바일 모두 지원
- WebGL, Three.js 3D 시각화 기술 데모

## 핵심 요구사항

- React + Vite + TypeScript 기반 프로젝트 구조
- Three.js를 활용한 6축 로봇 팔 3D 렌더링
- 계층적 관절 구조로 실제 로봇처럼 움직이는 구현
- 관절 각도 조절 UI (슬라이더 + 직접 입력)
- 좌표계 표시 (각 관절마다)
- 반응형 UI (데스크톱/모바일)
- 터치 제스처 지원
- 다크 테마 UI 스타일링

## 주요 특징

- 외부 모델 파일 없이 코드로 로봇 팔 생성
- Three.js 계층 구조를 활용한 정확한 관절 회전
- OrbitControls로 3D 뷰 조작
- 모바일 터치 제스처 지원
- 반응형 레이아웃
