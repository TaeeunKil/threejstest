# 프로젝트 개요

## 프로젝트 이름

rainbow-robotics-ui-demo

## 설명

레인보우로보틱스 UI/UX 프론트엔드 설계 및 개발자 면접 포트폴리오 프로젝트입니다.
Three.js를 사용하여 로봇 팔 시각화 UI를 구현합니다.

## 목표

- 레인보우로보틱스 면접 포트폴리오 제작
- Three.js를 활용한 6축 로봇 팔 3D 시각화 구현
- 실제 로봇 제어 UI와 유사한 인터페이스 개발
- WebGL, Three.js 등 3D 시각화 경험 증명

## 핵심 요구사항

- React + Vite 기반 프로젝트 구조
- Three.js를 활용한 6축 로봇 팔 3D 렌더링
- 관절 각도 조절 UI (슬라이더)
- 좌표계 표시 및 TCP 위치 표시
- 다크 테마 UI 스타일링
- 실제 레인보우로보틱스 UI와 유사한 디자인

## 프로젝트 배경

레인보우로보틱스 채용 공고 요구사항:

- 로봇 매니퓰레이터 제어 플랫폼의 프론트엔드 설계 및 개발
- 시뮬레이션/디버깅/제어/로직 구현 등을 위한 Web UI 설계 및 구현
- WebGL, Three.js 등 3D 시각화 경험 (우대조건)
