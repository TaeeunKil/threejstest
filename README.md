# Rainbow Robotics UI Demo

레인보우로보틱스 면접 포트폴리오 - Three.js 로봇 팔 시각화 프로젝트

## 기술 스택

- React 18
- Vite
- Three.js

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 프로젝트 구조

```
src/
├── App.jsx          # 메인 앱 컴포넌트
├── App.css          # 앱 스타일
├── main.jsx         # 진입점
└── index.css        # 전역 스타일
```
